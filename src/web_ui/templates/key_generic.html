Как получить ключ?
{% if service == 'shadowsocks' %}

Ключи Shadowsocks обычно предоставляются VPN-провайдерами в формате ссылки. Если у вас есть ссылка вида ss://..., просто скопируйте её целиком.

  • Метод шифрования: chacha20-ietf-poly1305, aes-256-gcm, и др.
  • Пароль: случайная строка
  • Сервер и порт: предоставляются провайдером
{% elif service == 'vless' %}

Ключи VLESS генерируются сервером Xray/V2Ray. Ссылка содержит UUID, настройки безопасности и транспорт.

  • UUID: уникальный идентификатор пользователя
  • Безопасность: tls, reality, none
  • Транспорт: tcp, ws, grpc
{% elif service == 'hysteria2' %}

Hysteria 2 — быстрый и устойчивый к блокировкам протокол на основе QUIC. Требует больше оперативной памяти и CPU.

  • Пароль: случайная строка (auth)
  • SNI: домен для TLS сертификата
  • obfs: optional obfuscation (sensitive)
  • obfspwd: пароль для обфускации
Внимание: Не рекомендуется для Keenetic KN-1210/1211/1212 (128MB RAM)
{% elif service == 'trojan' %}

Ключи Trojan предоставляются сервером Trojan. Ссылка содержит пароль и настройки TLS.

  • Пароль: случайная строка
  • TLS: обязательное шифрование
  • SNI: домен сервера
{% elif service == 'tor' %}

Мосты Tor можно получить:

  • На сайте bridges.torproject.org
  • В Telegram боте @GetBridgesBot
  • По запросу bridges@torproject.org (с Gmail)
{% endif %}
